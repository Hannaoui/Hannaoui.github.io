Introduction en HTML et CSS :

HTML est l’abréviation de HyperText Markup Language , soit en français « langage de balisage hypertexte ». Ce langage a été créé en 1991 et a pour fonction de structurer et de donner du sens à du contenu. Grâce au HTML, on va par exemple pouvoir indiquer au navigateur que tel texte doit être considéré comme un simple paragraphe ou que tel autre est un titre.

CSS est le diminutif de Cascading StyleSheets , ou feuilles de styles en cascade. Le CSS a été créé en 1996 et a pour rôle de mettre en forme du contenu en lui appliquant ce qu’on appelle des styles. Le CSS va nous permettre par exemple de définir la taille, la couleur ou l’alignement d’un texte. Nous allons donc utiliser le CSS sur notre code HTML, afin d’enjoliver le résultat visuel final.

Pour plus de details aller plus loin.

Le principe des balises en HTML :

Le point de départ du langage HTML repose sur l'utilisation de balises, elles seront délimitées par des chevrons : < ... >. Il existe trois types de balises :

  • Balise ouvrante : < balise >
  • Balise fermante : < / balise >
  • Balise auto-fermante : < balise / >, moins utilisée

tout ce qui se trouve entre deux balises définit un block. Et tout block est délimité entre une balise ouvrante et une balise fermante.

La structure d'un document en HTML :

Pour créer un document HTML , il faut respecter la structure suivante :

  • Tout document HTML doit commencer par : <!doctype html>.
  • Ensuite, l'ensemble du texte commence par < / html > et se terminera par < / html >.

Ensuite, le texte HTML se divise en plusieurs parties :

  • la tête du texte : < Head >. Cette partie contient les méta-informations du document :
  • le corps du texte : < body > corps du texte < / body >, qui va contenir le texte sémantique qui sera perçu par l'utilisateur futur (après transformation).
    • jeu d'encodage. On utilisera utf8.
    • titre du document borné par < title > ... et < / title >
  • header : représente un groupe de contenu introductif ou de contenu aidant à la navigation. Il peut contenir des éléments de titre, mais aussi d'autres éléments tels qu'un logo, un formulaire de recherche, etc. ( aller plus loin)
  • nav : représente une section d'une page ayant des liens vers d'autres pages ou des fragments de cette page. Autrement dit, c'est une section destinée à la navigation dans un document. ( aller plus loin)
  • aside : est une section de la page dont le contenu est indirectement lié à ce qui l'entoure et qui pourrait être séparé de cet environnement. ( aller plus loin)
  • footer : représente le pied de page de la section ou de la racine de sectionnement la plus proche. ( aller plus loin)
  • article : présente une section générique d'un document, par exemple un groupe de contenu thématique. Une section commence généralement avec un titre. ( aller plus loin)
  • section : représente une section générique d'un document, par exemple un groupe de contenu thématique. Une section commence généralement avec un titre. ( aller plus loin)

Les balises de traitement de texte en HTML:

  • <strong> : mise en valeur forte (exemple : HTML ).
  • <em> : mise en valeur normale autrement dit italique (exemple : HTML ).
  • <mark> : mise en valeur visuelle (exemple : HTML ).

Les balises interdites en HTML :

Ces balises appartiennet à une ancienne époque, donc ils est fortement conseillé de les éviter.

  • <br> : saut de ligne.
  • <hr> : grand trait.

Les listes en HTML :

  • <ol> : liste ordonnée.
  • <ul> : liste non-ordonnée.
  • <li> : ligne.

Les échappements en HTML :

Pour les besoins de rédaction, il se peut qu'on ait besoin d'écrire certains caractères comme < > ou &. On appelle cela l'échappement qui correspond à des normes d'écriture qui permettent au texte HTML de comprendre que es caractères sont à lire "normalement" et non pas pour définir un code précis. Ainsi,

  • pour le chevron < on écrira &lt;
  • pour le chevron > on écrira &gt;
  • pour le caractère & on écrira & amp;

Insertion de hyperliens en HTML

L'élément <a> définit un hyperlien vers un autre endroit de la même page ou vers une autre page sur le Web. il a un seul attribut obligatoire href, le mode d'emploie est le suivant :

<a href = "mon lien" > message </a>

Il contient plusieurs attributs qui ont chacun une fonction spécifique, à l'heure actuelle nous n'avons abordé que l'attribut target. Ce dernier attribut définit la fenêtre dans laquelle la page cible doit s'afficher : son utilisation permet donc de diriger l'ouverture d'un document cible vers une fenêtre déterrminée. Les valeurs qui peut prendre sont :

  • _self : (Valeur par défaut) ouverture de la page cible dans la meme page exemple.
  • _blank : Ouverture de la page cible dans une nouvelle fenêtre exemple.
  • _parent : Ouverture de la page cible dans le cadre parent ( de niveau immédiatement supérieur) exemple.
  • _top : Ouverture de la page cible dans la fenêtre hôte (par-dessus le FRAMESET) exemple.
  • _media : (disponible seulement avec IE 6 ou ultérieur) Ouverture de la page cible dans la zone HTLM de la "Media Bar" exemple.
  • _search : (disponible seulement avec IE 6 ou ultérieur) Ouverture de la page cible dans le volet d'exploration (rechercher) du navigateur exemple.
  • nom du cadre : Ouverture de la page cible dans le cadre portant le nom cité ( valeur de l'attribut name ) exemple.

La balise doit être accompagnée de deux attributs obligatoires :

  • src : il permet d'indiquer où se trouve l'image que l'on veut insérer. Vous pouvez soit mettre un chemin.
  • alt : cela signifie « texte alternatif ». On doit toujours indiquer un texte alternatif à l'image, c'est-à-dire un court texte qui décrit ce que contient l'image. Ce texte sera affiché à la place de l'image si celle-ci ne peut pas être téléchargée (cela arrive), ou dans les navigateurs de personnes handicapées (non-voyants) qui ne peuvent malheureusement pas « voir » l'image. Cela aide aussi les robots des moteurs de recherche pour les recherches d'images.

Les tableaux en HTML

La première balise à connaître est <table> </table>. Cette balise qui permet d'indiquer le début et la fin d'un tableau. Cette balise est de type bloc, il faut donc la placer en dehors d'un paragraphe.

les autres balises à connaître sont :

  • <tr> </tr> : indique le début et la fin d'une ligne du tableau,
  • <td> </td> : indique le début et la fin du contenu d'une cellule.

Schématiquement, un tableau se construit comme sur la figure suivante.

pour les en-têtes et les pieds de tableau on utilise les deux balises suivantes :

  • l'en-tête de tableau il se fdéfinit avec les balises suivates : <thead> </thead>
  • le pied de tableau il se fdéfinit avec les balises suivates : <tfoot> </tfoot>
    • <th> </th> : indique le début et la fin du contenu d'une cellule de l'en-tête ou du pied.

Pour fusionner des cellulles il y'a les deux attributs selon le sens de fusion :

  • rowspan : pour fusionner des cellules au sens horizontal
  • colspan : pour fusionner des cellules au sens vertical

l'utilisation est la suivante :

  • <td rowspan = "2" colspan ="2">

pour ajouter une légende à un tableau il faut utiliser la balise <caption> la légende </caption>.

enfin la balise colgroup qui définit un groupe de colonnes au sein d'un tableau. Pour cette balise on a exploré l'attribut span qui est un entier positif indiquant le nombre de colonnes consécutives sur lesquelles appliquer les attributs de l'élément. S'il est absent, sa valeur par défaut est 1.
