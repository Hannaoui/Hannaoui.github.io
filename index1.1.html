Introduction

HTML (HyperText Markup Language) est un langage de desciption de données et non de programation!

CSS (Cascading Style Sheets) est un langage utilisé pour la forme, qui nous permet d'inserer differents types d'éléments dans les pages web.

Pour plus de details cliquer ici.

Quelques balises de structuration de documents

  • <!doctype html> : Balise pour commencer un document.
  • <html> : Balise principale ouvrante
  • </html> : Balise principale fermante
  • <head> : En-tête de la page ouvrante
  • </head> : En-tête de la page fermante
  • <body> : Corps de la page ouvrant
  • </body> : Corps de la page fermant

Quelques balises de traitement de texte

  • <strong> : mise en valeur forte (exemple : HTML ).
  • <em> : mise en valeur normale autrement dit italique (exemple : HTML ).
  • <mark> : mise en valeur visuelle (exemple : HTML ).
  • <h1> mon titre </h1> : pour la taille des titres niveau 1 (le plus gros caractère).
  • <h2> mon titre </h2> : pour la taille des titres niveau 2.
  • <h3> mon titre </h3> : pour la taille des titres niveau 3.
  • <h4> mon titre </h4> : pour la taille des titres niveau 4.
  • <h5> mon titre < h5> : pour la taille des titres niveau 5.
  • <h6> mon titre </h6> : pour la taille des titres niveau 6 (le plus petit caractère).
  • <dl> ... </dl> : liste de définitions.
  • <dt> ... </dt> : terme à définir.
  • <dd> ... </dd> : définition des termes.

Le balises interdites

  • <br> : saut de ligne.
  • <hr> : grand trait.

Les Types de liste

  • <ol> : liste ordonnée.
  • <ul> : liste non-ordonnée.
  • <li> : ligne.

Echappement en HTML

Pour les besoins de rédaction, il se peut qu'on ait besoin d'écrire certains caractères comme < > ou &. On appelle cela l'échappement qui correspond à des normes d'écriture qui permettent au texte HTML de comprendre que es caractères sont à lire "normalement" et non pas pour définir un code précis. Ainsi,

  • pour le chevron < on écrira &lt;
  • pour le chevron > on écrira &gt;
  • pour le caractère & on écrira & amp;
